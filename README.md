## 📦 安装与运行指北

本程序需要在您的机器上安装 Python (建议 3.8 及以上版本)。

1. **安装增强版 Python 依赖（包括界面库）：**
   打开终端或命令提示符，进入本脚本所在目录并运行：
   ```bash
   pip install -r requirements.txt
   ```

2. **安装 Playwright 内核 (首次运行必须)：**
   ```bash
   playwright install chromium
   ```

3. **▶️ 启动图形界面桌面软件：**
   在终端运行：
   ```bash
   python gui_app.py
   ```
   *此时将弹出一个超级漂亮的深色UI界面，按照中文提示填入关键词点开始即可！*

## 🔨 如何将它打包成 .exe 卖钱？

有了 UI，就可以打包为独立的 exe 文件让别人没装 Python 也能用了。进入本目录运行：

```bash
3
```
打包成功后，会在 `dist` 目录下生成 `gui_app.exe`。
> **注意**：由于 Playwright 依赖 Chromium 内核打包进 exe 有点麻烦，如果你想卖给纯小白，建议连同 Python 环境与源码一起给他们，写个简单的一键运行批处理 `.bat`。或者使用更进阶的 PyInstaller hook 把 playwright 浏览器驱动包进去。
